# ❤️ My Favourite Rom-Coms

A React-based movie explorer application showcasing my favorite Telugu romantic comedy movies. Users can browse movies, view ratings, and explore detailed information about each film through a clean and responsive interface.

## 🚀 Features

* Home Page
* Movie Listing Page
* Movie Details Page
* Dynamic Routing using React Router
* Responsive Card Layout
* Reusable React Components
* Static Data Management
* Modern UI Design

## 🎬 Movies Included

* Pelli Choopulu
* Ante Sundaraniki
* Oohalu Gusagusalade
* Kushi
* Sammohanam

## 🛠️ Technologies Used

* React JS
* React Router DOM
* Vite
* JavaScript (ES6+)
* HTML5
* CSS3

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/myfav.git
```

Navigate to the project folder:

```bash
cd myfav
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```

## 🏗️ Build for Production

```bash
npm run build
```
